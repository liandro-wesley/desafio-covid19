----

Durante o treinamento da Hiring Coders, um desafio foi proposto:

- [x] Desenvolver sem auxílio de Frameworks uma página de notícias sobre o covid-19

- [x] A página precisa ter no menu, os seguintes links:
* Home
* Sobre
* Saiba mais
